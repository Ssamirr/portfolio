# Samir Sardarli — Portfolio

Personal portfolio website built with **React**, **Vite**, and **Tailwind CSS**.

🌐 **Live:** [samirsardarli.dev](https://samirsardarli.dev)

## Tech Stack

- React 19
- Tailwind CSS 3
- Vite
- Vercel Analytics

## Features

- Responsive dark theme design
- Animated hero with typewriter effect
- Skills, Experience, Education, Contact sections
- Deployed on Vercel with custom domain
